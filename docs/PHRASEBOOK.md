## Transliteration hints

- `he` -> `ሀ`
- `hu` -> `ሁ`
- `hi` -> `ሂ`
- `ha` -> `ሃ`
- `hee` -> `ሄ`
- `h` -> `ህ`
- `ho` -> `ሆ`
- `hua` -> `ኋ`
- `ae` -> `አ`
- `au` -> `ኡ`
- `ai` -> `ኢ`
- `aa` -> `ኣ`
- `aee` -> `ኤ`
- `a` -> `እ`
